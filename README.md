# sebodeppo # hesap_makinesi

Bu proje, C# kullanılarak geliştirilmiş basit bir konsol tabanlı
hesap makinesi uygulamasıdır.

## Özellikler
- Toplama
- Çıkarma
- Çarpma
- Bölme
- Hatalı giriş kontrolü

## Kullanılan Teknolojiler
- C#
- .NET Console Application

## Amaç
Programlama temellerini, metot kullanımını ve kontrol yapılarını
pekiştirmek.
